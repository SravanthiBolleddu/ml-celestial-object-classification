# Machine Learning Based Classification of Celestial Objects

## Overview
This project applies machine learning techniques to classify celestial objects such as galaxies, stars, and quasars using astronomical survey data.

## Methodology
- Data preprocessing and feature selection
- Random Forest classification model
- Train-test split for evaluation

## Features Used
- Redshift
- Photometric bands (u, g, r, i, z)
- Spatial coordinates (alpha, delta)

## Results
- Achieved ~97.87% accuracy
- Evaluated using confusion matrix

- Python
- Pandas, NumPy
- Scikit-learn
- Matplotlib
